# Inferential Statistics Practical

This repository contains practical learning work on inferential statistics using Python. The project focuses on understanding statistical testing, hypothesis testing, data analysis, and interpretation of results using real dataset practice.

## Project Overview

Inferential statistics helps us make conclusions about a population using sample data. This repository includes hands-on practice with statistical concepts such as Z Test, T Test, Chi Square Test, confidence intervals, hypothesis testing, and data-based decision making.

The main objective of this project is to build practical understanding of statistical analysis instead of only learning formulas theoretically.

## Key Concepts Explained

Z Test

Z Test is used when the sample size is large and population standard deviation is known. It helps compare sample mean with population mean.

T Test

T Test is used when the sample size is small or population standard deviation is unknown. It is commonly used to compare means.

Chi Square Test

Chi Square Test is used for categorical variables. It helps check whether two categorical variables are associated with each other.

P Value

P value helps decide whether the result is statistically significant. A small p value usually means there is enough evidence to reject the null hypothesis.

Confidence Interval

Confidence interval gives a range of values where the true population parameter is likely to lie.
